# Brainfu — интерпретатор Brainfuck на Python

Простой интерпретатор языка Brainfuck с поддержкой интерактивного режима и загрузки программ из файлов.

## Возможности

- Выполнение всех базовых команд Brainfuck: `+`, `-`, `>`, `<`, `.`, `,`
- Интерактивный режим (ввод команд с клавиатуры)
- Загрузка программ из `.bf`-файлов
- Команды управления:  
  - `exit` — завершить работу;  
  - `clear` — сбросить все ячейки и вывод;  
  - `info` — показать информацию.
- Вывод состояния:  
  - значения всех ячеек;  
  - накопительный вывод (`self.output`);  
  - номер текущей ячейки.
